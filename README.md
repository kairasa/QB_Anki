# QB_Anki
医学生用問題集QBのAnkiカードを作る短縮ツール
QBの問題から解説までをコピーアンドペーストだけで自動でいらない部分を消去、表面と裏面に問題・解答解説を分ける、見やすいUIに変更、Anki Connect経由で自動追加までできます。
手動ですが画像もQBからコピペできるので画像問題も対応しています。
国試・CBT対応
分野ごとのタグ付けもできます。

導入方法
1.Windows版Ankiを以下のサイトからインストールする
https://apps.ankiweb.net/#downloads

2.AnkiにAnkiConnectをダウンロードする。
Anki内のツール　→　アドオン　→　新たなアドオンを取得　→　2055492159と入力
詳しくは公式サイト参照
https://ankiweb.net/shared/info/2055492159

3.Ankiデッキを作成
デッキ名は”国試”もしくは”CBT”にしてください

4.qb_to_anki.exeファイルを起動する

使用方法
0.Anki、qb_to_anki.exeを起動
1.デッキ名を国試・CBTから選択する
2.任意でタグ付けする科目を選択する
3.問題文から解説の終わりでコピーして”問題を貼り付け”にペーストする
4.解析ボタンをクリック
5.任意で画像を貼り付け
6.Ankiに追加をクリック
